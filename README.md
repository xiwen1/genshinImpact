## Django初体验之：咒术回站

#  一部伟大的史诗级巨著！！ 

## 年轻人的第一个web项目

### 登陆游戏请使用网址：http://114.55.29.194:8000/

### 玩法说明：
1. 鼠标右键点击移动
2. 鼠标左键释放技能
3. f键选中技能"逆焰之刃"，受到攻击的敌人获得火附着，技能cd1s，可充能三次；
4. d键选中技能"霜袭"，受到攻击的敌人获得冰附着，技能cd1.5s，可充能两次；
5. "逆焰之刃"累计命中敌人两次后，下一次f键技能转换成元素爆发："黎明"；
6. "逆焰之刃"以及"黎明"对冰附着的敌人造成的伤害翻倍，击退距离翻倍；
